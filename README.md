# Query Generation

1. TPC-DS 
2. TPC-H 
3. JOB

In two ways 
1. Snowflake
2. Naive

# Main program
For documentation you may run
```bash
pixi run main --help
```
# Testing
To run the test you may use
```bash
pixi run test
```
